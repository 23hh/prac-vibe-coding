# Vibe Coding 연습 프로젝트

바이브코딩 연습을 위한 HTML, CSS, JavaScript 데모 프로젝트 모음입니다.

## 📚 프로젝트 목록

### 1. Todo Firebase App
Firebase Realtime Database를 활용한 할일 관리 애플리케이션

- **기술 스택**: HTML5, CSS3, JavaScript (ES6+), Firebase Realtime Database
- **주요 기능**: 할일 추가/수정/삭제, 실시간 동기화, 필터링
- **위치**: `todo-firebase/`

### 2. Vibe Shop
쇼핑몰 데모 애플리케이션

- **기술 스택**: HTML5, CSS3, JavaScript
- **위치**: `Vibe-shop/`

## 🎯 학습 목표

- HTML/CSS/JavaScript 기초 실습
- Firebase를 활용한 실시간 데이터베이스 연동
- 모던 웹 개발 패턴 학습
- 반응형 웹 디자인 구현

## 🚀 시작하기

각 프로젝트 폴더의 README.md 파일을 참고하여 실행하세요.

```bash
# 예시: Todo Firebase App 실행
cd todo-firebase
# Live Server 또는 로컬 서버 실행
```
