# Vibe Coding 練習プロジェクト

バイブコーディング練習のためのHTML、CSS、JavaScriptデモプロジェクト集です。

## 📚 プロジェクト一覧

### 1. Todo Firebase App
Firebase Realtime Databaseを活用したタスク管理アプリケーション

- **技術スタック**: HTML5, CSS3, JavaScript (ES6+), Firebase Realtime Database
- **主な機能**: タスク追加/編集/削除、リアルタイム同期、フィルタリング
- **場所**: `todo-firebase/`

### 2. Vibe Shop
ショッピングモールデモアプリケーション

- **技術スタック**: HTML5, CSS3, JavaScript
- **場所**: `Vibe-shop/`

## 🎯 学習目標

- HTML/CSS/JavaScript基礎実習
- Firebaseを活用したリアルタイムデータベース連携
- モダンウェブ開発パターンの学習
- レスポンシブウェブデザインの実装

## 🚀 始め方

各プロジェクトフォルダのREADME.mdファイルを参照して実行してください。

```bash
# 例: Todo Firebase App実行
cd todo-firebase
# Live Serverまたはローカルサーバーを実行
```

## 📝 注意事項

- 各プロジェクトは独立して実行可能です
- Firebaseを使用するプロジェクトは設定ファイルが必要です
- 詳細は各プロジェクトのREADME.mdを参照してください
